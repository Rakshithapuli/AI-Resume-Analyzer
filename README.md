# AI Resume Analyzer

AI-powered Resume Analyzer built using Python, Streamlit, and Google Gemini API.

## Features

- Upload Resume PDF
- Paste Job Description
- Analyze Resume
- Match Percentage
- Missing Skills Detection
- Resume Improvement Suggestions

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- PyPDF2

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run application

streamlit run app.py
